# Lithology Prediction Using Machine Learning

This project applies multiple machine learning models to predict subsurface lithology from well log measurements.

## 📊 Dataset
Well log measurements including:
- SP, GR, LLD, LLS, DEN, AC
- Depth

Target:
- Lithology classification

## 🧠 Models
- Linear Regression
- Decision Tree
- Logistic Regression
- Convolutional Neural Network (CNN)

## 📈 Results
- Linear Regression: ~93.7%
- CNN: ~76.8%
- Decision Tree: ~70.7%
- Logistic Regression: ~57.7%

## 🎯 Goal
To explore how machine learning can be used in geoscience for subsurface prediction.
